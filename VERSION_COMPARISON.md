# 버전 비교: 기본 vs 업그레이드

프로젝트에는 **두 가지 버전**이 있습니다. 필요에 따라 선택하세요!

---

## 📊 버전 비교표

| 특징 | 기본 버전 | 업그레이드 버전 (v2) |
|------|-----------|---------------------|
| **파일명** | `index.html` | `index_v2.html` |
| **CSS** | `styles.css` | `styles_v2.css` |
| **JS** | `script.js` | `script_v2.js` |
| **지도** | 간단한 SVG | SVG + 이미지 지원 |
| **별자리** | SVG 그리기 | SVG + 이미지 지원 |
| **완료 화면** | CSS 별 효과 | CSS + 천상열차 이미지 |
| **애니메이션** | 기본 | 강화된 효과 |
| **파일 크기** | 작음 | 중간 (이미지 포함 시) |
| **로딩 속도** | 빠름 | 중간 |
| **커스터마이징** | 중간 | 높음 |

---

## 🎯 어떤 버전을 선택해야 할까요?

### ✅ 기본 버전 (`index.html`)을 선택하세요

**이런 경우에 추천:**
- 빠른 프로토타이핑이 필요할 때
- 이미지 에셋 준비가 어려울 때
- 가벼운 파일 크기가 중요할 때
- SVG 스타일이 충분히 만족스러울 때
- 빠른 로딩 속도가 필요할 때

**장점:**
- ⚡ 즉시 실행 가능 (추가 파일 불필요)
- 🚀 빠른 로딩
- 📦 작은 용량
- 🎨 깔끔한 SVG 스타일
- ✨ 모든 핵심 기능 동일

**단점:**
- 실제 지도 이미지 미사용
- 별자리가 기하학적 패턴

---

### ✅ 업그레이드 버전 (`index_v2.html`)을 선택하세요

**이런 경우에 추천:**
- 더 풍부한 시각적 경험을 원할 때
- 실제 한반도 지도를 사용하고 싶을 때
- 천상열차분야지도 이미지가 있을 때
- 커스텀 별자리 디자인을 사용하고 싶을 때
- 더 화려한 애니메이션이 필요할 때

**장점:**
- 🖼️ 실제 이미지 사용 가능
- ✨ 강화된 애니메이션 효과
- 🎨 더 풍부한 시각적 표현
- 🔧 높은 커스터마이징 가능성
- 📸 더 사실적인 표현

**단점:**
- 이미지 준비 필요 (선택사항)
- 약간 큰 파일 크기
- 약간 느린 로딩 (이미지 사용 시)

---

## 🔄 버전 전환 방법

### 기본 → 업그레이드로 전환

1. 브라우저에서 `index_v2.html` 열기
2. (선택) 이미지 에셋 추가
3. `script_v2.js`에서 `CONFIG` 설정 변경

### 업그레이드 → 기본으로 전환

1. 브라우저에서 `index.html` 열기
2. 끝! (모든 기능 동일하게 작동)

## 🎨 시각적 차이

### 지도 렌더링

**기본 버전:**
```
• 간단한 SVG 패스
• 청록색 아웃라인
• 기본 발광 효과
• 정적인 라인
```

**업그레이드 버전:**
```
• 실제 지도 이미지 또는 향상된 SVG
• 더 디테일한 발광
• 맥박 애니메이션 (pulse)
• 데코레이션 별들
```

### 별자리 표현

**기본 버전:**
```
• SVG 도형으로 그리기
• 단순한 선과 점
• 기본 그리기 애니메이션
```

**업그레이드 버전:**
```
• 이미지 또는 향상된 SVG
• 강화된 발광 효과
• 반짝임 애니메이션
• 커스텀 디자인 가능
```

### 핫스팟 (지역 표시)

**기본 버전:**
```
• 40×40px 크기
• 기본 펄스 효과
• 별(★) 아이콘
```

**업그레이드 버전:**
```
• 50×50px 크기 (더 큼)
• 회전 + 바운스 효과
• 향상된 발광
• 방문 시 특별한 애니메이션
```

## 🔧 혼합 사용도 가능!

업그레이드 버전에서 **일부만 이미지 사용**할 수 있습니다:

```javascript
// script_v2.js 설정 예시
const CONFIG = {
    useKoreaMapImage: true,        // 지도만 이미지
    useConstellationImages: false, // 별자리는 SVG
    useCheonsangImage: false,      // 완료 화면은 CSS
};
```

이렇게 하면 **최상의 균형**을 찾을 수 있습니다!
